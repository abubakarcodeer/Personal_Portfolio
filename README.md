# Modern Professional Portfolio | AbuBakar Codeer

A high-end, interactive developer portfolio built with a focus on immersive user experience, clean aesthetics, and modern web engineering.

![Portfolio Preview](public/hospital%20Queue%20Managment%20system.webp)

## 🚀 Technical Highlights

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS 4 (Custom Midnight & Cyan Theme)
- **3D Visuals:** Three.js via React Three Fiber & Drei
- **Animations:** Framer Motion (Scroll Reveals, Magnetic Interactions, Bento Transitions)
- **Icons:** React Icons (Silicon, Bootstrap, DevIcons)
- **Deployment:** Vercel

## ✨ Key Features

- **Immersive 3D Hero:** A glassmorphic, interactive 3D sphere that responds to mouse movement and distorts in real-time.
- **Bento Grid Projects:** Apple-inspired non-linear layout for project showcase.
- **Spotlight Cards:** Interactive cards with a subtle cyan glow following the cursor for a premium feel.
- **Advanced Micro-interactions:** Magnetic buttons that "pull" toward the user's cursor.
- **Comprehensive Tech Stack:** Showcasing expertise across Frontend, Backend, Database, and AI tools.
- **Responsive Architecture:** Perfectly optimized for Mobile, Tablet, and Desktop experiences.
